## Dataset

The dataset folder should be like this:

```
datasets
├── grefcoco
│   ├── grefs(unc).json
│   ├── instances.json
└── images
    └── train2014
        ├── COCO_train2014_xxxxxxxxxxxx.jpg
        ├── COCO_train2014_xxxxxxxxxxxx.jpg
        └── ...
```
Download our gRefCOCO from [here](https://github.com/henghuiding/gRefCOCO)!
